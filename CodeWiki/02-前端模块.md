# 前端模块详解

## 1. Electron 主进程（app/main.js）

### 模块职责
主进程是整个应用的入口与控制中心，负责窗口管理、系统集成、子进程调度及安全策略。

### 关键常量与配置

| 常量 | 说明 |
|------|------|
| `defaultSettings` | 默认用户设置对象，覆盖下载路径、线程数、Cookie、同步上限、主题、快捷键、云同步凭据等。 |
| `SENSITIVE_COOKIE_NAMES` | 需要脱敏的 Cookie 键名集合。 |
| `SYNC_MAX_RUNTIME_MS` | 同步任务最大运行时长（10 分钟）。 |
| `SYNC_STALL_MS` | 同步任务无输出卡死阈值（3 分钟）。 |

### 核心类/函数

#### `findPythonExecutable(): Promise<string|null>`
- **职责**：自动发现可用的 Python 解释器。
- **策略**：
  1. 检查缓存的 `cachedPythonPath`。
  2. 读取用户设置中的 `pythonPath`。
  3. 依次尝试 `py -3`、`python`、`python3` 及大量 Windows 常见绝对路径。
  4. 确认该解释器可导入 `aiohttp`（douyin-downloader 的依赖）。
- **返回**：可执行路径或 `null`。

#### `createWindow()`
- 创建无边框主窗口（`frame: false, titleBarStyle: 'hidden'`）。
- 尺寸 1280x840，最小 1000x640，暗色背景 `#0b0d14`。
- 加载 `renderer/index.html`，开发环境自动打开 DevTools。

#### `createTray()` / `updateTrayMenu()` / `toggleMainWindow()`
- 创建系统托盘图标（支持 PNG/ICO）。
- 托盘菜单提供“显示/隐藏主窗口”与“退出”。
- macOS 下支持 Dock 图标设置。

#### `registerGlobalShortcuts()` / `unregisterGlobalShortcuts()`
- 注册全局快捷键：粘贴下载（`Ctrl+Shift+V`）、切换窗口（`Ctrl+Shift+D`）、暂停全部（`Ctrl+Shift+P`）。
- 设置保存后自动重新注册。

#### `runPythonBridge(scriptPath, taskId, jobFile, cwd, progressChannel, logChannel, finishedChannel, activeMap)`
- **通用 Python 桥接启动器**，被下载、博主作品、新发布、关系、报表、字幕、云同步等复用。
- 流程：
  1. 调用 `findPythonExecutable()`。
  2. `spawn` Python 子进程，传入 `--job` 与 `--task-id`。
  3. stdout 按行解析 JSON；解析成功发 `progressChannel`，失败发 `logChannel`。
  4. 进程关闭后删除 job 文件，发送 `finishedChannel`。

#### `validateCookiesWithBackend(cookieString): Promise<object>`
- 将 Cookie 写入临时文件，调用 `douyin-downloader/login_service.py` 校验登录态。
- 返回 `{ valid: boolean, user?: object, reason?: string }`。

#### `auth:loginWithBrowser`
- 打开独立的登录窗口（960x760），加载 `https://www.douyin.com/`。
- 注入反检测脚本（覆盖 `navigator.userAgent`、`navigator.webdriver`）。
- 拦截非抖音导航/重定向/弹窗，防止跳转到外部应用（如 `bitbrowser://`）。
- 页面加载完成后注入“我已完成登录”按钮；点击后提取 Cookie 并后端校验。

#### 敏感信息脱敏函数
- `redactText(text)`：对手机号、Cookie、Authorization、API Key 脱敏。
- `redactObject(obj)`：递归对对象中敏感键脱敏。
- **用途**：所有发给渲染进程的日志与 IPC 数据均先经过脱敏。

#### 进程管理
- `trackProcess(proc)`：将子进程加入集合，便于退出时统一清理。
- `killProcessTree(pid)`：Windows 使用 `taskkill /T /F`；Unix 递归 SIGTERM 子进程树。
- `cleanupAllChildProcesses()`：应用退出前强制结束所有追踪的子进程。

## 2. 预加载脚本

### app/preload.js
通过 `contextBridge.exposeInMainWorld('electronAPI', {...})` 暴露主窗口所需的全部 IPC API，分为：
- 窗口控制：`minimize`, `maximize`, `close`, `isMaximized`
- 设置：`getSettings`, `setSettings`
- 目录/档案：`selectFolder`, `openFolder`, `listArchive`, `deleteArchive`, `openVideo`
- 各类任务启动/取消：`startDownload`, `cancelDownload`, `startSync`, `cancelSync`, ...
- 各类事件监听：`onDownloadProgress`, `onDownloadLog`, `onDownloadFinished`, ...

### app/login-preload.js
仅暴露 `completeLogin`，供登录窗口内的“我已完成登录”按钮调用。

## 3. 渲染进程（app/renderer/app.js）

### 技术特点
- 使用 Vue 3 **全局构建版**（`Vue.createApp`），无单文件组件（SFC），所有组件以对象形式定义在 `app.js` 中。
- 使用 Vue 的 ` reactive / ref / computed / watch / provide / inject` 进行状态管理。
- 暗色主题通过 `document.documentElement.setAttribute('data-theme', 'dark')` 切换。

### 全局 Store

```javascript
const store = reactive({
  settings: { ...defaultSettings },   // 用户设置
  tasks: [],                          // 下载任务列表
  archive: [],                        // 作品档案（本地文件夹）
  syncs: [],                          // 同步任务列表
  relationTasks: [],                  // 关系操作任务
  reportTasks: [],                    // 报表任务
  transcriptTasks: [],                // 字幕任务
  cloudTasks: [],                     // 云同步任务
  userWorks: [],                      // 博主作品任务
  newReleases: { status, items, ... },// 新发布检测状态
  syncCache: { favorites, likes, following }, // 同步缓存数据
  currentPage: 'downloads',           // 当前页面路由
  initialized: false,                 // 是否已加载设置
  authChecked: false,                 // 登录态是否已校验
  effectiveTheme: 'dark',
  user: { isLoggedIn, nickname, avatar, sec_uid, unique_id },
});
```

### Store 方法

| 方法 | 说明 |
|------|------|
| `loadSettings()` | 从主进程读取设置，合并默认值，校验登录态，加载档案与同步缓存。 |
| `checkAuth()` | 用当前 Cookie 调用后端校验，更新 `store.user`。 |
| `login(cookieString)` / `loginWithBrowser()` | 保存 Cookie 并重新校验登录态。 |
| `saveSettings()` | 将 `store.settings` 写回主进程。 |
| `applyTheme()` | 强制应用暗色主题（当前仅支持 dark）。 |

### 页面路由

前端通过 `store.currentPage` 控制页面切换，已实现的页面包括：

| 页面标识 | 说明 | 状态 |
|----------|------|------|
| `downloads` | 下载中心（输入链接、批量下载、任务列表） | 完整实现 |
| `following` | 关注列表（搜索、排序、筛选、分页、批量操作） | 完整实现 |
| `favorites` | 收藏列表 | 接入导航，功能由同步+档案支撑 |
| `collections` | 合集列表 | 接入导航 |
| `likes` | 喜欢列表 | 接入导航 |
| `archive` | 作品档案（本地视频文件夹浏览） | 完整实现 |
| `tasks` | 任务中心（各类任务历史与状态） | 完整实现 |
| `newReleases` | 新发布发现 | 完整实现 |
| `userWorks` | 博主作品列表 | 完整实现 |
| `reports` | 报表导出 | 完整实现 |
| `transcripts` | 字幕生成 | 完整实现 |
| `cloud` | 云同步（备份/恢复） | 完整实现 |
| `settings` | 设置（下载、同步、快捷键、云、关于） | 完整实现 |

### 公共工具函数

| 函数 | 说明 |
|------|------|
| `generateAvatar(name)` | 根据名字首字母生成彩色 SVG 头像。 |
| `formatTimeLabel(iso)` | 将 ISO 时间转为“刚刚/x分钟前/x小时前”等中文相对时间。 |
| `extractSecUidFromUrl(url)` | 从抖音用户主页 URL 提取 `sec_uid`。 |
| `formatFileSize(bytes)` | 字节转为 B/KB/MB/GB/TB。 |
| `formatDownloadLog(line)` | 将 Python 输出的 JSON 日志格式化为中文可读文本。 |
| `plain(obj)` | 深拷贝为普通对象（解决 Vue reactive proxy 无法通过 IPC 序列化的问题）。 |
| `translateFilenameTemplate(template)` | 将中文模板变量转为英文占位符（如 `{日期}` -> `{date}`）。 |

### 图标库
所有图标采用内联 SVG（Feather 风格线框图标），定义在 `icons` 对象中，包含 user、bookmark、download、layers、clock、folder、settings、search、refresh、grid、list 等。

## 4. 样式（app/renderer/style.css）

- 以 CSS 变量定义暗色主题色板（背景、表面、边框、主色、成功、警告、错误等）。
- 支持滚动条美化、无边框窗口的自定义标题栏、响应式布局。
- 动画：淡入、滑入、脉冲、旋转等。
