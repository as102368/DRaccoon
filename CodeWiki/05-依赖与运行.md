# 依赖关系与项目运行方式

## 1. 前端依赖

### Electron 运行时
- 项目不通过 npm 安装 Electron，而是依赖预编译二进制放在相邻目录 `electron/`。
- `start.bat` 直接调用 `electron\electron.exe app` 启动。

### 第三方库（渲染进程）
- **Vue 3.4+**（全局构建版）：`app/renderer/vendor/vue.global.js`，离线可用，无需 npm 构建。
- 无其他前端构建依赖（Vite/Webpack 均未使用）。

### Node.js 内置模块使用
- `electron`：窗口、IPC、托盘、对话框、剪贴板、全局快捷键、原生图片。
- `path`、`fs`、`child_process`：文件与进程操作。
- `os`：获取用户目录、平台判断。

## 2. Python 依赖

### requirements.txt

| 包名 | 版本 | 用途 |
|------|------|------|
| `openpyxl` | >=3.1.0 | 报表导出 Excel |
| `boto3` | >=1.34.0 | S3 兼容云存储 |
| `oss2` | >=2.18.0 | 阿里云 OSS |
| `cryptography` | >=42.0.0 | 云备份 Fernet 加密 |
| `openai-whisper` | >=20231117 | 本地语音识别 |
| `opencc-python-reimplemented` | >=0.1.7 | 中文简繁转换 |

### 外部项目依赖（运行时动态导入）
- **douyin-downloader**：必须存在于 `../../douyin-downloader` 目录（相对于 `python/lib/compat.py`）。
- 该项目额外需要的 Python 包（由 douyin-downloader 决定，常见包括）：
  - `aiohttp`（HTTP 客户端）
  - `rich`（CLI 终端输出，被 `DummyDisplay` 屏蔽）
  - 数据库相关：`aiosqlite` 或 `sqlite3`
  - 其他内部工具库

> 主进程 `findPythonExecutable()` 会显式检查目标解释器能否导入 `aiohttp`，以确保 douyin-downloader 可用。

## 4. 项目运行方式

### Windows（推荐）

直接双击项目根目录的 `start.bat`：

```batch
@echo off
start "" "%~dp0electron\electron.exe" "%~dp0app"
```

该脚本会启动 Electron 并加载 `app/` 目录。

### 手动运行（开发调试）

```bash
# 前提：electron 二进制在 PATH 中，或指定绝对路径
electron ./app
```

开发环境下，主进程会检测 `NODE_ENV === 'development'` 并自动打开 DevTools。

### Python 环境准备

1. 安装 Python 3.10+，并确保 `aiohttp` 等依赖已安装：
   ```bash
   pip install -r python/requirements.txt
   ```
2. 将 `douyin-downloader` 项目放置在本项目的同级目录：
   ```
   /workspace/
   ├── douyin-downloader/   <-- 放在这里
   └── ...
   ```
3. （可选）在应用设置中指定 `pythonPath`，避免自动探测失败。

## 5. 数据与配置存储

| 类型 | 路径 | 说明 |
|------|------|------|
| 用户设置 | `<userData>/settings.json` | Electron `app.getPath('userData')` 下 |
| 任务文件 | `<userData>/jobs/*.json` | 运行时临时任务参数，结束后自动删除 |
| 日志 | `<userData>/logs/` | Electron 日志目录 |
| 下载目录 | `~/Downloads/DouyinDownloaded`（默认） | 视频、封面、音乐等下载内容 |
| 数据库 | `<downloadDir>/dy_downloader.db` | 下载历史、关注列表等 SQLite 数据库 |
| Cookie 文件 | `<downloadDir>/.cookies.json` | 持久化 Cookie |
| 同步缓存 | `<downloadDir>/.sync/{kind}.json` | 收藏/喜欢/关注同步缓存 |

## 6. 关键外部依赖关系图

```
DRaccoon (本项目)
    ├── Electron Runtime (electron/)
    │       └── 加载 app/ (main.js -> renderer/)
    ├── Python Bridges (python/*_bridge.py)
    │       ├── lib/bridge.py
    │       ├── lib/redactor.py
    │       ├── lib/compat.py  ──> 添加 ../../douyin-downloader 到 sys.path
    │       └── 运行时导入 douyin-downloader 的模块：
    │               - auth.CookieManager
    │               - cli.main / cli.report
    │               - core.api_client / core.relation_service / core.audio_extraction
    │               - storage.Database
    │               - utils.cookie_utils / utils.logger / utils.helpers
    │               - config.ConfigLoader / control.RateLimiter
    └── 外部服务（可选）
            ├── OpenAI API (字幕转录)
            ├── S3 / OSS (云备份)
            └── 抖音 Web/API (业务数据)
```

> 若缺少 `douyin-downloader`，所有 Python 桥接脚本在导入阶段即会失败，主进程会在日志中报告“找不到模块”。
