# DRaccoon / Douzy 项目概览

## 项目简介

DRaccoon（也称 Douzy）是一款基于 **Electron + Vue 3 + Python** 的抖音内容管理桌面端应用。它提供了视频下载、关注列表同步、批量关注/取关、作品档案管理、字幕生成、报表导出及云同步等能力。

项目采用前后端分离的混合架构：
- **前端**：Electron 提供桌面窗口与系统级能力，Vue 3（CDN 引入）构建用户界面。
- **后端**：Python 桥接脚本通过子进程方式调用，利用外部 `douyin-downloader` 内核完成抖音 API 请求、下载与数据处理。

## 目录结构

```text
/workspace/
├── app/                          # Electron 应用源码
│   ├── main.js                   # 主进程入口（窗口、IPC、Python 调度）
│   ├── preload.js                # 主窗口预加载脚本（暴露 IPC API）
│   ├── login-preload.js          # 登录窗口预加载脚本
│   ├── package.json              # Electron 应用元数据
│   └── renderer/                 # 渲染进程前端
│       ├── index.html            # 页面入口
│       ├── app.js                # Vue 3 应用逻辑（组件、状态、页面）
│       ├── style.css             # 暗色主题样式
│       ├── test-following.html   # 关注列表测试页
│       └── vendor/
│           └── vue.global.js     # Vue 3 全局构建版（CDN 离线）
├── python/                       # Python 桥接层
│   ├── lib/                      # 公共库
│   │   ├── __init__.py
│   │   ├── bridge.py             # 桥接基类与 JSON Lines 协议
│   │   ├── redactor.py           # 敏感信息脱敏工具
│   │   └── compat.py             # 兼容层（引入 douyin-downloader）
│   ├── download_bridge.py        # 下载桥接
│   ├── relation_bridge.py        # 批量关注/取关桥接
│   ├── report_bridge.py          # 报表导出桥接
│   ├── transcript_bridge.py      # 字幕生成桥接
│   ├── cloud_bridge.py           # 云备份/恢复桥接
│   ├── user_works_bridge.py      # 博主作品列表桥接
│   ├── new_releases_bridge.py    # 新发布发现桥接
│   └── requirements.txt          # Python 依赖
├── README.md                     # 项目说明
├── start.bat                     # Windows 启动脚本
└── check_state.py                # WebSocket CDP 状态检查脚本
```

## 技术栈

| 层级 | 技术 |
|------|------|
| 桌面壳 | Electron (v31.0.2) |
| 前端框架 | Vue 3（全局构建版，CDN 离线引入） |
| 样式 | 纯 CSS（暗色主题） |
| 后端语言 | Python 3.10+ |
| 核心依赖 | douyin-downloader（相邻目录，动态引入） |
| 通信 | Electron IPC + stdout JSON Lines |

## 核心功能列表

- **视频下载**：支持单条/批量抖音链接下载，含封面、音乐、头像等。
- **同步**：收藏、喜欢、关注列表同步至本地 SQLite。
- **博主作品**：拉取指定博主作品元数据（不下载文件）。
- **新发布**：基于已同步关注列表，检测关注博主的新作品。
- **批量关系操作**：批量关注/取关。
- **报表导出**：按日期/作者/模式聚合，导出 Excel/HTML。
- **字幕生成**：支持 OpenAI API 或本地 Whisper，输出 SRT/TXT/VTT/JSON。
- **云同步**：加密备份设置与数据库到 S3 或阿里云 OSS。
- **全局快捷键**：粘贴下载、切换窗口、暂停全部。
